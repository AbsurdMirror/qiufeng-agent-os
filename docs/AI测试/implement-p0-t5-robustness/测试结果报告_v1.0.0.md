- 现状：`record()` 仅返回 NormalizedRecord，仓库内未见统一的“record -> JSONLStorageEngine.write_record”落盘调用点。
- 影响：如果不通过测试脚本显式写入 JSONL，`cli_logger` 将无法观察到全链路日志，T5 的 OB-P0-04/05/06 难以在真实运行中兑现。
- 建议：在 App/OE 层补齐观测写盘接入点（此项属于业务实现，不在本测试职责范围内修改）。
